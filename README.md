## Stack

- **Frontend**: Vanilla JavaScript (no frameworks). Web Crypto API for crypto.
- **Backend**: PHP/MySQL minimalist (directory only — tools run client-side).
- **Hosting**: cPanel/LiteSpeed + Cloudflare.

## Why this exists

Most online calculators apply US/UK legislation, require signup, and have invasive ads. [haz.tools](https://haz.tools/?utm_source=github&utm_medium=web2&utm_campaign=backlinks2026) solves all three:

- Spanish legislation, updated to 2026 (RDL 16/2025 for self-employed tax, etc.)
- Client-side execution (data never leaves the browser)
- No signup, no invasive ads
